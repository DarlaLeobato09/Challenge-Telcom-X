# Telecom X - Análisis de Evasión de Clientes

## Descripción
Este proyecto analiza la **evasión de clientes (churn)** de Telecom X.  
El objetivo es identificar patrones de cancelación y generar insights que ayuden a diseñar estrategias de retención.

## Datos
- Formato: JSON
- Contiene información de clientes, incluyendo género, tipo de contrato, método de pago, servicios contratados y gastos mensuales.
- Campos clave analizados: `Churn`, `Gender`, `Contract`, `PaymentMethod`, `Charges.Monthly`, `PhoneService`, `StreamingTV`, `StreamingMovies`, entre otros.

## Herramientas
- **Python 3**  
- Bibliotecas: `pandas`, `numpy`, `matplotlib`, `seaborn`, `json_normalize`  
- Entorno: Google Colab
